Store notebook outputs here, organized by notebook section.

- `artifacts/04_feature_engineering/figures`
- `artifacts/04_feature_engineering/tables`
- `artifacts/05_model_preprocessing/figures`
- `artifacts/05_model_preprocessing/tables`
